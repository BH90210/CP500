# CP500
Social media automation app for Android

Build Go from source
Assuming everything is in default location run in your terminal
`gomobile bind -o $GOPATH/src/github.com/BH90210/CP500/AndroidStudioProject/app/cpfiveoo.aar -target=android github.com/BH90210/CP500/cpfiveoo`
